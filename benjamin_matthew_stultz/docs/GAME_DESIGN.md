# Benji's Obby - Game Design Document

## Overview
A classic obstacle course (Obby) game where players navigate through challenging stages to reach the finish line.

## Core Features

### 1. Checkpoint System
- Players respawn at the last checkpoint they touched
- Checkpoints are clearly marked with glowing pads
- Progress is saved per player session

### 2. Stage System
- Multiple stages of increasing difficulty
- Each stage has unique obstacles
- Stage counter shows current progress

### 3. Death Zones
- Falling off platforms resets to last checkpoint
- Hazard blocks (lava, spikes) cause instant death
- Kill bricks are clearly colored (red/orange)

### 4. Timer System
- Track how long it takes to complete the obby
- Personal best times displayed
- Optional: Leaderboard for fastest times

## Stage Ideas

### Stage 1-5: Easy (Tutorial)
- Basic jumps
- Wide platforms
- No moving parts

### Stage 6-10: Medium
- Smaller platforms
- Moving platforms
- Simple timing challenges

### Stage 11-15: Hard
- Precise jumps
- Fast moving obstacles
- Combination challenges

### Stage 16-20: Expert
- Tiny platforms
- Multiple hazards
- Speed sections

## Color Coding Guide
- **Green**: Safe/Checkpoint
- **Red/Orange**: Danger/Kill brick
- **Blue**: Moving platform
- **Yellow**: Bounce pad
- **White/Gray**: Normal platform

## Controls
- WASD: Movement
- Space: Jump
- Shift: Sprint (optional)
